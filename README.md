Tic-Tac-Toe Game

A console-based Tic-Tac-Toe game implemented in Python for two players.
Features
- Two-player interactive game
- Board displayed in the console
- Win detection logic
- Uses lists, loops, and functions

Technologies Used
- Python

Purpose
This project was developed to practice logical problem solving and implementation of game logic using basic Python concepts.

 How to Use
1. Clone or download the repository  
2. Run the Python file in a Python environment  
3. Follow on-screen prompts to play the game

Learning Outcomes
- Working with lists and functions
- Implementing game rules
- Handling user input
